## Installation

To install the desktop application, head to the [Releases](https://github.com/nsbuitrago/rma-kinetics-app/releases) section and download
the appropriate version for your operating system. Follow the system prompts
to complete installation.

## Updating

Updates are checked when the application first launches. You can manually
check for updates from the side bar if needed. If an update is available, you can
choose to download and install the latest version directly from within the application.
